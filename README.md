## 🦀 dgui (desktop GUI)

A Simple, Modular, and Highly Customizable Reactive Retained Mode Desktop GUI framework.

> [!WARNING]
> ⚠️ Note: DGUI is currently in active development and not recommended for production use.

### 🎯 Purpose & Goal

`dgui` is lightweight, flexible and highly customizable.

- 🎮 For both desktop tools and in-game HUDs.
- ✨ Layout, styling are completely customizable.
- 🚀 Easy to use API, while maintaining 60fps.
- 🔀 Native Node Graph Support.
