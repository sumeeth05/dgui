# 🖥️ DGUI : Desktop GUI

A simple, modular, and highly customizable immediate mode Desktop GUI framework.

[![Crates.io](https://img.shields.io/crates/v/dgui.svg?color=orange)](https://crates.io/crates/dgui)
[![GitHub](https://img.shields.io/github/v/release/sumeeth05/dgui?color=blue&label=github)](https://github.com/sumeeth05/dgui)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/sumeeth05/dgui/blob/main/LICENSE)

> [!WARNING]
> ⚠️ Note: DGUI is currently in active development.

## 🎯 Purpose & Goal

DGUI is designed to be a lightweight, flexible and highly customizable.

- 🎮 For both desktop tools and in-game HUDs.
- ✨ Layout, styling are completely customizable.
- 🚀 Keeps the API minimal while using fast, modern Rust crates underneath.
- 🔀 Native Node Graph Support.
